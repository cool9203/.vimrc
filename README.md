# vimrc
## how to use
### step 1 : install vim-plug
```shell
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

### step 2 : copy ./.vimrc to ~/.vimrc
```shell
cp ./.vimrc ~/.vimrc
```

### step 3 : pluginstall
```
#in vim
:PlugInstall
:PlugStatus
```
then restart vim

### step 4 : ./run.sh
```
./run.sh
```
